## datetime型とtimestamp型
- datetimeを使う(文字列扱い)
- 作成日にはcurrent_timestampをデフォルト値につける
- データを挿入すると自動で`2022-04-05 05：51：57`が入る(ver5.7で確認。verによって違うかも)
- 更新日には手動で日付を作成して挿入する
  
phpの場合はこう
```php
date('Y-m-d H：i：s')
// 2022-04-05 05：51：57
```
datetime型は文字列なので何でも入る。(多分)
同じ形式の文字列を作成して挿入する。
