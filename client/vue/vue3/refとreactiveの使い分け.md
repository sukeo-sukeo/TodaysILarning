# ref
プリミティブな値に使う
|||
|-|-|
|使い所| プリミティブな値に使う|
|script内| hoge.valueでアクセス|
|template内| hogeでアクセス|
|ポイント| 値がオブジェクトの場合、プロパティを追加してもリアクティブに変更されない|


# reactive
値がオブジェクトのときに使う
|||
|-|-|
|使い所| オブジェクトに使う|
|script内| hoge.valueでアクセス|
|template内| hoge.valueでアクセス|
|ポイント| 追加したプロパティもリアクティブに変更される|


# shallowRef
コンポーネントオブジェクトのときに使う
- <component> タグで動的に切り替えるときに使用
